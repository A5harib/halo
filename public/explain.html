4. Architecture Walkthrough & Pipelines

ProspectHalo operates as a serverless Next.js application leveraging edge-ready API routes connecting to Groq. By utilizing the openai/gpt-oss-120b model, response times are extremely rapid.

Key Build Pipeline:

Dev Mode: Turbopack transpiles globals.css and parses React components. Next.js 16 compiles and bundles assets. By importing external fonts via layout headers instead of CSS files, we bypass CSS ordering parsing rules and improve Page Speed metrics.
